# v4.0.0 Upgrade Guide
This document will provide information on upgrading from Terra Dev Site `^3.0.0` to `4.0.0`.

## Dependency Changes

Terra Dev Site was updated to use terra-base `v5`.

Reference [terra-base's upgrade guide](https://engineering.cerner.com/terra-ui/#/components/terra-base/base/upgrade-guide) for changes needed to consume this Terra Dev Site upgrade.

# Package Loader

The Package Loader will take the passed in json file and return a set of react components including a badge component with a customizable url.
The file must be a json file.

#### Note: 
The `github source` badge will only be displayed if the json file contains the custom property `src` in the following format:
```jsx
"src": {
    "url": "https://github.com/cerner/terra-dev-site"
  }
```

## Resource query

`?dev-site-package`

## Badge Props

Prop | type
--- | ---
url | string

## Badge Example

```jsx
import { Badge } from './pretendPackage.json?dev-site-package';

<Badge />

<Badge url="https://www.google.com"/>
```

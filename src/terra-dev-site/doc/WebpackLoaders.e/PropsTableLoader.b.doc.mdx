# Props Table Loader

The props table loader will take the passed in file and return a react component representing the props information for that component. The props table loader will attempt to find the `src` file fro any `lib` file passed in. The props table can only be generated for non transpiled files and will fail if the file passed in has been transpiled and no file of the same name can be found in the `src` directory.

## Resource query

`?dev-site-props-table`

## Props Table Example

```jsx
import PropsTable from './PropsTableExample?dev-site-props-table';

<PropsTable />

```

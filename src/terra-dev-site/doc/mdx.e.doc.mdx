# MDX

Terra dev site uses MDX as a markdown loader. MDX allows you to mix JSX syntax into your markdown documents.

For example the following block will render into

```md
# Hello, *world*!

Below is an example of JSX embedded in Markdown. <br /> **Check out
the background color!**

<div style={{ padding: '20px', backgroundColor: 'tomato' }}>
  <h3>This is JSX</h3>
</div>
```

With the output below:
<hr />

# Hello, *world*!

Below is an example of JSX embedded in Markdown. <br /> **Try and change
the background color!**

<div style={{ padding: '20px', backgroundColor: 'tomato' }}>
  <h3>This is JSX</h3>
</div>

<hr />

Just make sure you close your tags.

For more information see [https://mdxjs.com/](https://mdxjs.com/)
